How Mingo Browser & Hosting Works

This guide explains how to use the Mingo Browser to host, view, and manage websites on the Mingo platform.

What is Mingo Browser?

Mingo Browser is a built-in web environment that allows users to open, preview, and manage websites hosted on the Mingo platform without using external browsers.

Mingo Browser is designed for learning, testing, and hosting static websites built with HTML, CSS, and JavaScript.

How Mingo Browser Works

  1. You create and upload a website file(HTML, CSS, JS).
  2. The website code is saved inside the Mingo platform.
  3. Mingo Browser loads the website using an internal preview system.
  4. You can access the site using a Mingo domain.

How to Host a Website on Mingo

  1. Prepare your website files (HTML, CSS, JavaScript).
  2. Open the Hosting section inside Mingo.
  3. Upload or your website file.
  4. Choose a domain name ending with: .sele , .mingo , .dent , .dev.
Supported Domains

  1. Save and host your website.
  2. Your website becomes instantly available.

Managing Hosted Websites

Once hosted, you can:

  • Edit your website code at any time.
  • Delete hosted websites.
  • Preview changes instantly in Mingo Browser.

Current Limitations

Mingo currently supports static websites only. Backend technologies such as databases and server-side scripts are not yet supported.
